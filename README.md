# Liri-the-personal-assistant
Using node js and multiple api's 'Liri' is the best CLI assistant around!

## Make it so liri.js can take in one of the following commands:

   * `concert-this`

   * `spotify-this-song`

   * `movie-this`

   * `do-what-it-says`
